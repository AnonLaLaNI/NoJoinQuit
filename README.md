# NoJoinQuit

A clean and lightweight Minecraft plugin that removes default join and quit chat messages.

> Silent server chat. No spam. No setup.

## Features
- Removes **join messages** (`PlayerJoinEvent`)
- Removes **quit messages** (`PlayerQuitEvent`)
- No commands
- No permissions
- No config required

## Compatibility
- Paper API: **1.21**
- Works on Paper and compatible Spigot-based servers

## Installation
1. Download `NoJoinQuit-1.0.jar`
2. Put it in your server's `/plugins/` folder
3. Restart the server

## Build from source
```bash
mvn clean package
```
Output:
`target/NoJoinQuit-1.0.jar`
